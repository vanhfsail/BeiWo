## 技术点

* 1.Bmob的免费云服务后台
* 2.Universal-ImageLoader的图片加载
* 3.Mob的免费短信SDK
* 4.自定义的SwipeRefreshLayout
* 5.circleImageView圆形图像
* 6.图片缓存处理
* 7.SQLite数据存储
